# Moni Monitor

Moni Monitor is a lightweight microfinance wallet and loan management system
built on Google Apps Script with a secure customer web interface.

## Structure

moni-monitor/
├── backend/        # Google Apps Script backend (Code.gs)
├── frontend/       # Customer-facing HTML UI
├── README.md
└── .gitignore

## Features
- Customer wallet
- Secure PIN authentication
- Transaction ledger
- Loans & repayments
- Audit trail
- Daily automated backups

## Deployment
Backend code is deployed via Google Apps Script.
Frontend HTML is served via HtmlService.
